This plugin integrates Feedback Company reviews and the WooCommerce connector into a Wordpress website.  Dutch language only.

== Description ==

This plugin requires an API key from Feedback Company to function.  Using an API key, you can display widgets with customer reviews on your Wordpress website.  You can choose from several widgets.  For WooCommerce users, it's also possible to enable the WooCommerce connector, which automatically asks users to review your product(s) and/or store.

After installation of the plugin, you will find an overview of the widgets and corresponding shortcodes on the settings screen.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/the-feedback-company` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Obtain an API key from Feedback Company
4. Use the Settings->Feedback Company screen to configure the plugin
5. Add the widget to your website via Appearance or the shortcodes.
